Detailed Description

Implementation of bin packing algorithms.

REFERENCE: Approximation algorithms for bin packing: A survey

  • First fit (ff) - implemented
  • First fit decreasing (ffd) - implemented
  • Best fit (bf) - not yet implemented
  • Best fit decreasing (bfd) - not yet implemented
  • Worst fit (wf) - not yet implemented
  • Worst fir decreasing (wfd) - not yet implemented

void packing::ff ( vector< double > &  trafficToPack,
list< router > &  r_list 
)

First Fit bin packing algorithm.

forward a flow using the first available backend router and the first available link on that router which have a capacity to handle the flow. If such a router is not available, drop the flow.

Parameters:
trafficToPack input traffic for packing
r_list list of routers to pack
Returns:
r_list list of routers PROCESS:
  • forward a flow using the first available router and the first link available on that router that can handle the flow
  • update different flow statistics such as residual capacity of a currently used link, lost flows, objective value, etc., on the way

if flow fits into a link, compute the residual capacity of both the router and the link
