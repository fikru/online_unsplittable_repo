\subsection{\-Detailed \-Description}
sort routers and links according to their efficiency; also used to sort the input traffic 

\-After reading the list of routers from a text configuration file, the algorithm sorts the routers according to their efficiency if \-\_\-\-R\-O\-U\-T\-E\-R\-\_\-\-P\-O\-W\-E\-R\-\_\-\-S\-O\-R\-T is defined.

\-Procedure\-: \begin{DoxyItemize}
\item sort the routers according to (only router power considered for sorting the routers)\-: \begin{eqnarray} \eta_{_{NIC-}} = \frac{C_{r}}{P_{r}} \end{eqnarray} \-O\-R (both link and router power considered for sorting the routers) \begin{eqnarray} \eta_{_{NIC+}} = \frac{C_{r}}{P_{r} + \sum_{i=1}^N{P_{rl}}} \end{eqnarray} \item sort the links on each router according to\-: \begin{eqnarray} \eta_{_l} = \frac{C_{rl}}{P_{rl}} \end{eqnarray}
